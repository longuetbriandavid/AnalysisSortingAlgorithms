Interface graphique :
- Les checkbuttons s'appelant "Afficher le graphe x" servent également a
	enregistrer ces graphes. 
	Pour simplement afficher un graphe, il faut sélectionner la case correspondante.
	Lorsque l'on souhaite enregistrer un graphe il faut également sélectionner la case puis sélectionner
		une des cases d'enregistrement appropriées ( avec/sans affichage ). 


Pour lancer les tests il faut que vous modifier la variable choice dans le fichier main.py à -1, le lancement de l'algo à 2 et la visualisation pygame vous mettez n'importe quelle valeur.

Si vous voulez changer d'algorithme il faut commenter l'ancien et vous décommentez le nouveau.
